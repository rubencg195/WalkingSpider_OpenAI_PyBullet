================================================================================
                            TROUBLESHOOTING
================================================================================

Problem: "error: Microsoft Visual C++ 14.0 or greater is required"
Solution: Use Conda setup (setup_environment_conda.bat)
Time: 5-10 minutes

Problem: "ModuleNotFoundError: No module named 'gym'"
Solution: Ensure virtual environment is activated
  Windows: venv_spider\Scripts\activate
  Unix: source venv_spider/bin/activate

Problem: PyBullet import fails after installation
Solution: Run test_pybullet_simple.py for diagnostics
Then: Check SETUP_AND_DEBUG_REPORT.md for your OS

Problem: GUI doesn't appear during testing
Solution: This is okay, use headless mode
  Set: render=False in env creation
  GIFs will still be saved to videos/ folder

More help: See README.md "Troubleshooting" section
